Data Types, Operators, Expressions & Variables

Data Type

A data type specifies the type of data that a variable can store. It determines how much memory is allocated and what operations can be performed on the variable.

Common Data Types:
int, float, double, char

Variable

A variable is a named storage location in memory that holds a value. The value can change during program execution.

Example:
int age;

Operator

An operator is a symbol that tells the compiler to perform specific mathematical or logical operations.

Examples:
+, -, *, /, %

Expression

An expression is a combination of variables, constants, and operators that produces a single value when evaluated.

Example:
a + b * 2

Example: All Concepts Together

int a = 5;          // Variable 'a' of type int
int b = 10;         // Variable 'b' of type int
int result;         // Variable 'result' (uninitialized)

result = a + b * 2; // Expression using operators (+, *)
                    // Result: 5 + (10 * 2) = 25

printf("Result: %d", result);
Explanation:
  • int is the data type
  • a, b, result are variables
  • +, * are operators
  • a + b * 2 is an expression

Variable Naming Rules

When creating variable names in C, you must follow specific rules:

Must start with a letter or underscore

Variable names can begin with any letter (a-z, A-Z) or an underscore (_)

age, _count, totalScore
Can contain letters, digits, and underscores

After the first character, you can use letters, numbers, and underscores

age2, student_name, total123
Case-sensitive

C distinguishes between uppercase and lowercase letters

Age, age, AGE are all different
Cannot start with a digit

Variable names cannot begin with numbers

✗ 2students, 9count
Cannot use reserved keywords

Keywords like int, if, else, while cannot be used as variable names

✗ int, float, return, for
No spaces or special characters

Spaces and most special characters (except underscore) are not allowed

✗ my age, total$, student#1

Constants and Variable Declarations

Constants

A constant is a value that cannot be changed during program execution. Constants are defined using the #define directive or the const keyword.

#define PI 3.14159        // Using #define
const int MAX_SIZE = 100; // Using const keyword

Variable Declarations

There are several ways to declare and initialize variables in C:

1 Uninitialized Declaration

Declared a variable without assigning an initial value. The variable contains garbage value.

int x;        // x is only declared but not initialized
float price;  // price is only declared but not initialized
2 Initialized Declaration

Declare and assign a value to a variable in one statement. This can be called initialized value.

int age = 25;       // age is initialized to 25
float pi = 3.14;    // pi is initialized to 3.14
3 Multiple Variables Declared

Declare multiple variables of the same type in one statement.

int a, b, c;           // Three variables declared
float x, y, z;
4 Multiple Initialization

Declare and initialize multiple variables in one statement.

int a = 5, b = 10, c = 15;       // All initialized
float x = 1.5, y = 2.5, z = 3.5;

Complete Example: All Declaration Types

int main() {
    // Uninitialized
    int count;
    
    // Initialized age is 21
    int age = 21;
    
    // Multiple variables declared
    int x, y, z;
    
    // Multiple initialization
    int a = 1, b = 2, c = 3;
    
    // Constant
    const float PI = 3.14159;
    
    return 0;
}

if (a != b)  // true, executes block
    printf("a is not equal to b");

Logical Operators

Logical operators combine boolean expressions and return a boolean result:

&& Logical AND

Returns true (1) if both operands are true.

int a = 5, b = 10;

if (a > 0 && b > 0)  // true AND true = true
    printf("Both positive");

if (a > 10 && b > 5) // false AND true = false
    printf("Won't print");
|| Logical OR

Returns true (1) if at least one operand is true.

int age = 16;

if (age < 13 || age > 19) // false OR false = false
    printf("Not a teenager");

if (age > 15 || age < 10) // true OR false = true
    printf("Will print");
! Logical NOT

Returns the opposite boolean value (inverts true to false, false to true).

int isRaining = 0; // false

if (!isRaining)        // NOT false = true
    printf("Go outside!");

if (!(5 > 10))         // NOT false = true
    printf("5 is not greater than 10");

Increment and Decrement Operators

These operators increase or decrease a variable's value by 1:

++ Increment

Increases the value by 1

int x = 5;
x++;      // x becomes 6
// Same as: x = x + 1;

-- Decrement

Decreases the value by 1

int y = 10;
y--;      // y becomes 9
// Same as: y = y - 1;

Prefix vs Postfix

The placement of the operator (before or after the variable) affects when the increment/decrement happens:

Prefix (++x, --x) Increment FIRST

The variable is incremented/decremented before its value is used in the expression.

int x = 5;
int y = ++x;  // x becomes 6 first, then y = 6

printf("x = %d, y = %d", x, y);
// Output: x = 6, y = 6
Postfix (x++, x--) Use FIRST

The variable's current value is used in the expression first, then it is incremented/decremented.

int x = 5;
int y = x++;  // y = 5 first, then x becomes 6

printf("x = %d, y = %d", x, y);
// Output: x = 6, y = 5

Side-by-Side Comparison:

// PREFIX
int a = 5;
int b = ++a;  // a = 6, b = 6

// POSTFIX
int x = 5;
int y = x++;  // x = 6, y = 5

Type Conversion (Casting)

Type conversion allows you to convert a value from one data type to another. There are two types:

Implicit Conversion (Automatic)

The compiler automatically converts one data type to another when needed. This usually happens when mixing different types in an expression.

int a = 5;
float b = 2.5;
float result = a + b;  // 'a' is automatically converted to float
                       // 5.0 + 2.5 = 7.5

printf("%.1f", result); // Output: 7.5

Note: C promotes smaller types to larger types (int → float → double) to prevent data loss.

Explicit Conversion (Manual Casting)

You manually convert a value to a specific type using the cast operator. Syntax: (type) value

float x = 7.8;
int y = (int) x;  // Explicitly convert float to int
                  // Truncates decimal: 7.8 → 7

printf("%d", y);  // Output: 7

// Another example
int a = 5, b = 2;
float result = (float) a / b;  // 5.0 / 2 = 2.5
// Without cast: 5 / 2 = 2 (integer division)

Note: Casting from float to int truncates (removes) the decimal part, it does not round.

Comparison Example:

// Without casting (integer division)
int result1 = 7 / 2;        // 3 (both are int)

// With casting (float division)
float result2 = (float) 7 / 2;  // 3.5

Compound Statement

A compound statement (also called a block) is a group of statements enclosed in curly braces { }. It allows you to group multiple statements together so they are treated as a single unit.

// Single statement (no braces needed)
if (x > 0)
    printf("Positive");

// Compound statement (braces required)
if (x > 0) {
    printf("x is positive\n");
    printf("Value: %d\n", x);
    x = x * 2;
}
Best Practice

Always use curly braces { }, even for single statements in if/else/while/for blocks. This prevents bugs when you later add more statements.

Complete Example:

int score = 85;

if (score >= 50) {
    // Compound statement - multiple lines treated as one block
    printf("You passed!\n");
    printf("Your score: %d\n", score);
    score += 10;  // Bonus points
    printf("With bonus: %d\n", score);
}

Writing Simple C Programs

Formatting Output with printf()

The printf() function allows you to control how data is displayed using format specifiers, field width, precision, and flags.

Field Width

The field width specifies the minimum number of characters to be printed. If the value has fewer characters, it is padded with spaces.

int num = 42;

printf("%d\n", num);    // Output: 42
printf("%5d\n", num);   // Output:    42  (3 spaces + 42)
printf("%10d\n", num);  // Output:         42  (8 spaces + 42)
Explanation:
  • %5d means the output should be at least 5 characters wide
  • Since 42 is only 2 digits, 3 spaces are added before it
  • The number is right-aligned by default

Precision for Floating-Point Numbers

Use .n to specify how many decimal places to show:

float pi = 3.14159265;

printf("%f\n", pi);      // Output: 3.141593 (default 6 decimals)
printf("%.2f\n", pi);    // Output: 3.14
printf("%.4f\n", pi);    // Output: 3.1416 (rounded)
printf("%10.2f\n", pi);  // Output:       3.14 (width 10, precision 2)

Flags

Flags modify the output format:

Meaning
Example

Complete Example:

int num = 42;
float price = 19.99;

printf("%5d\n", num);      //    42
printf("%-5d\n", num);     // 42   
printf("%05d\n", num);     // 00042
printf("%+d\n", num);      // +42
printf("$%6.2f\n", price); // $ 19.99
✓ Benefits of Format Specifiers:
  • Create aligned, professional-looking output
  • Control decimal precision for accurate financial calculations
  • Format tables and reports with proper spacing
  • Make output more readable and consistent

Comments

Comments are notes in your code that are ignored by the compiler. They help explain what the code does, making it easier for you and others to understand.

// Single-Line Comment

Starts with // and continues to the end of the line.

// This is a single-line comment
int age = 21;  // Declare and initialize age

// Calculate area of rectangle
int area = length * width;
/* */ Multi-Line Comment

Starts with /* and ends with */. Can span multiple lines.

/*
 * This is a multi-line comment.
 * It can span several lines.
 * Useful for longer explanations.
 */
int calculate(int x, int y) {
    /* This is also valid */
    return x + y;
}
When to Use Comments
  • Explain complex logic or algorithms
  • Describe what a function does
  • Mark sections of code for organization
  • Temporarily disable code during testing
  • Add reminders or to-do notes

Functions in stdio.h

The <stdio.h> header file provides functions for input and output operations. The two most commonly used functions are:

printf() Output Function

Prints formatted output to the screen (standard output).

// Syntax:
printf("format string", arguments);

// Examples:
printf("Hello, World!");
printf("Age: %d", 25);
printf("Name: %s, Score: %.2f", "Alice", 95.5);
Common Format Specifiers:
  • %d or %i - Integer
  • %f - Float/Double
  • %c - Single character
  • %s - String
  • %lf - Double (for scanf)
scanf() Input Function

Reads formatted input from the keyboard (standard input).

// Syntax:
scanf("format string", &variable);

// Examples:
int age;
scanf("%d", &age);  // Note the & (address-of operator)

float price;
scanf("%f", &price);

char grade;
scanf(" %c", &grade);  // Space before %c to skip whitespace
Important: Use & with scanf()

For variables (not arrays), you must use the & operator before the variable name in scanf(). This passes the memory address where the input should be stored.

Complete C Programs

Program 1: Print a Line of Text

#include <stdio.h>

int main() {
    printf("Welcome to C Programming!\n");
    printf("This is my first program.\n");
    
    return 0;
}
Program Explanation:
  1. #include <stdio.h> - Includes the standard input/output library, which provides printf()
  2. int main() - The main function where program execution begins. int means it returns an integer
  3. printf(...) - Prints the text to the screen. \n creates a new line after each message
  4. return 0; - Returns 0 to the operating system, indicating the program finished successfully
  5. { } - Curly braces define the body (block) of the main function
Output: 
Welcome to C Programming!
This is my first program.

Program 2: Adding Two Integers

#include <stdio.h>

int main() {
    int num1, num2, sum;
    
    // Prompt user for input
    printf("Enter first number: ");
    scanf("%d", &num1);
    
    printf("Enter second number: ");
    scanf("%d", &num2);
    
    // Calculate sum
    sum = num1 + num2;
    
    // Display result
    printf("\nThe sum of %d and %d is %d\n", num1, num2, sum);
    
    return 0;
}
Program Explanation:
  1. int num1, num2, sum; - Declares three integer variables to store the two numbers and their sum
  2. printf("Enter...") - Prompts the user to enter a number
  3. scanf("%d", &num1) - Reads an integer from the keyboard and stores it in num1. The & gives scanf the memory address where to store the value
  4. sum = num1 + num2; - Adds the two numbers and stores the result in sum
  5. printf(..., num1, num2, sum) - Displays the result. %d placeholders are replaced by the variable values
Sample Run: 
Enter first number: 15
Enter second number: 27

The sum of 15 and 27 is 42

Key Takeaways

  • Data types specify what kind of data a variable can store (int, float, char, double)
  • Variables must start with a letter or underscore, are case-sensitive, and cannot be reserved keywords
  • Arithmetic operators: + - * / % (modulus gives remainder)
  • Relational operators (>, >=, <, <=, ==, !=) compare values and return true/false
  • Logical operators (&&, ||, !) combine boolean expressions
  • Prefix (++x) increments before use; Postfix (x++) uses value then increments
  • Compound assignment: x += 5 is the same as x = x + 5
  • Operator precedence: *, /, % before +, -; use parentheses for clarity
  • Implicit conversion is automatic; explicit conversion uses (type) casting
  • Format specifiers control output: %5d for width, %.2f for precision
  • Escape sequences: \n (newline), \t (tab), \\ (backslash), \" (quote)
  • printf() outputs data; scanf() reads input (use & for variables)
  • Every C program needs #include <stdio.h> and int main()
