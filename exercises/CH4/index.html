Control Structures

Learn how to control program flow using selection and iteration structures

14 Topics 15 Exercises ~45 min

Types of Control Structures

A control structure determines the order in which statements are executed in a program. There are three fundamental types:

Sequence Structure

Statements are executed one after another in the order they appear. This is the default flow of execution.

Example:
statement1;
statement2;
statement3;

Selection Structure

Allows the program to choose between different paths based on a condition. Implemented using if, if-else, and switch statements.

Methods:
if, if-else, switch

Iteration Structure

Allows statements to be executed repeatedly (loops) as long as a condition is true. Implemented using for, while, and do-while loops.

Methods:
for, while, do-while

Sequence Structure

The sequence structure is the simplest control structure. Statements are executed sequentially, one after another, in the exact order they are written in the program.

#include <stdio.h>

int main() {
    int a = 5;           // Step 1: Declare and initialize a
    int b = 10;          // Step 2: Declare and initialize b
    int sum = a + b;     // Step 3: Calculate sum
    printf("Sum: %d\n", sum);  // Step 4: Display result
    
    return 0;
}
Flow of Execution:

Each statement executes from top to bottom without any branching or repetition. The program starts at the first statement and proceeds linearly until it reaches the end.

Part 1

Selection Structures

Selection Statements in C

Selection statements allow your program to make decisions and execute different code based on conditions. There are three types:

if Simple Condition

Executes a block of code only if a condition is true. If the condition is false, the code is skipped.

if-else Two-Way Decision

Executes one block of code if the condition is true, and a different block if the condition is false.

switch Multi-Way Decision

Selects one of many blocks of code to execute based on the value of an expression. Useful when checking multiple possible values.

Selection: if Statement

The if statement executes a block of code only if a specified condition is to true (non-zero).

Condition
TRUE ↓ Execute Code
FALSE ↓ Skip Code

Example:

int age = 20;

if (age >= 18) {
    printf("You are an adult.\n");
    printf("You can vote!\n");
}

printf("End of program");

// Output:
// You are an adult.
// You can vote!
// End of program
Explanation:
  • The condition age >= 18 is evaluated
  • Since 20 >= 18 is true, the code inside the braces executes
  • Both printf statements run
  • If age was less than 18, the entire block would be skipped

Selection: if-else Statement

The if-else statement provides an alternative path of execution. If the condition is true, the if-block executes; otherwise, the else-block executes.

Condition
TRUE ↓ Execute if-block
FALSE ↓ Execute else-block

Example:

int score = 75;

if (score >= 50) {
    printf("Congratulations! You passed.\n");
    printf("Your score: %d\n", score);
} else {
    printf("Sorry, you failed.\n");
    printf("Better luck next time!\n");
}

// Output: (since 75 >= 50 is true)
// Congratulations! You passed.
// Your score: 75
Explanation:
  • The condition score >= 50 is checked
  • If true (score is 50 or more), the if-block executes
  • If false (score is less than 50), the else-block executes
  • Exactly one of the two blocks will always run

else-if Ladder (Multiple Conditions)

You can chain multiple conditions using else if:

int grade = 85;

if (grade >= 90) {
    printf("Grade: A\n");
} else if (grade >= 80) {
    printf("Grade: B\n");
} else if (grade >= 70) {
    printf("Grade: C\n");
} else if (grade >= 60) {
    printf("Grade: D\n");
} else {
    printf("Grade: F\n");
}

// Output: Grade: B

Selection: switch Statement

The switch statement allows you to select one of many code blocks to execute based on the value of a variable or expression. It's cleaner than multiple if-else statements when checking for specific values.

Important Points:
  • break exits the switch after a case executes
  • Without break, execution falls through to the next case
  • default is optional and executes if no case matches
  • The expression must evaluate to an integer or character

Example:

int day = 3;

switch (day) {
    case 1:
        printf("Monday\n");
        break;
    case 2:
        printf("Tuesday\n");
        break;
    case 3:
        printf("Wednesday\n");
        break;
    case 4:
        printf("Thursday\n");
        break;
    case 5:
        printf("Friday\n");
        break;
    case 6:
        printf("Saturday\n");
        break;
    case 7:
        printf("Sunday\n");
        break;
    default:
        printf("Invalid day\n");
}

// Output: Wednesday
Explanation:
  • The variable day has the value 3
  • The switch checks each case until it finds case 3
  • printf("Wednesday\n"); executes
  • break; exits the switch statement
  • If day was 10, the default case would execute

Switch with Characters:

char grade = 'B';

switch (grade) {
    case 'A':
        printf("Excellent!\n");
        break;
    case 'B':
        printf("Good job!\n");
        break;
    case 'C':
        printf("Satisfactory\n");
        break;
    case 'D':
        printf("Needs improvement\n");
        break;
    case 'F':
        printf("Failed\n");
        break;
    default:
        printf("Invalid grade\n");
}

// Output: Good job!

Null Statement

A null statement (also called an empty statement) is a statement that consists of only a semicolon ;. It does nothing but satisfy the syntax requirement when a statement is needed.

// Null statement in an if
if (x > 10)
    ;  // Null statement - does nothing

// Null statement in a loop
while (getchar() != '\n')
    ;  // Keep reading until newline
Common Mistake with Null Statements:

Accidentally placing a semicolon after an if statement creates a null statement, which can lead to logic errors:

Valid Use of Null Statement:

// Wait until a condition becomes true
while (sensorReady() == 0)
    ;  // Null statement - just wait

// Now proceed after sensor is ready
processData();
Part 2

Iteration Structures (Loops)

Iteration Structure

The iteration structure (also called a loop) allows you to execute a block of code repeatedly as long as a specified condition remains true. This eliminates the need to write the same code multiple times.

Why Use Loops?

  • Avoid Repetition: Write code once, execute it many times
  • Process Collections: Iterate through arrays, lists, or data structures
  • Handle Unknown Quantities: Loop until a condition is met.
  • Perform Calculations: Sum numbers, find averages, search for values

C provides three types of loops: for, while, and do-while. Each has different use cases, but all can accomplish the same tasks.

for Loop

The for loop is used when you know in advance how many times you want to repeat a block of code. It's ideal for counter-controlled loops.

Syntax:
for (initialization; condition; update) {
    // Code to repeat
}
1. Initialization

Executes once at the beginning. Usually initializes a counter variable.

int i = 0
2. Condition

Checked before each iteration. Loop continues while true.

i < 5
3. Update

Executes after each iteration. Usually increments the counter.

i++

Step-by-Step Execution:

1
Initialization: i = 0 executes once
2
Check condition: Is i < 5? If true, continue. If false, exit loop.
3
Execute body: Run the code inside the loop
4
Update: i++ executes (i becomes 1)
Repeat: Go back to step 2

Example: Print Numbers 1 to 5

for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

// Output:
// 1
// 2
// 3
// 4
// 5

Example: Sum of First 10 Numbers

int sum = 0;

for (int i = 1; i <= 10; i++) {
    sum = sum + i;  // or: sum += i;
}

printf("Sum = %d\n", sum);

// Output: Sum = 55
Trace Through the Loop:
  • Iteration 1: i=1, sum = 0+1 = 1
  • Iteration 2: i=2, sum = 1+2 = 3
  • Iteration 3: i=3, sum = 3+3 = 6
  • ... continues until i=10, sum = 45+10 = 55

while Loop

The while loop repeats a block of code as long as a specified condition is true. The condition is checked before each iteration, so the loop body may never execute if the condition is initially false.

Syntax:
while (condition) {
    // Code to repeat
}

Step-by-Step Execution:

1
Check condition: Is the condition true?
2
If TRUE: Execute the loop body
3
If FALSE: Exit the loop immediately
Repeat: Go back to step 1

Example: Print Numbers 1 to 5

int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;  // Don't forget to update!
}

// Output:
// 1
// 2
// 3
// 4
// 5
Avoid Infinite Loops:

Always make sure the loop condition will eventually become false. Forgetting to update the loop variable creates an infinite loop:

Example: Input Validation

int num;

printf("Enter a positive number: ");
scanf("%d", &num);

while (num <= 0) {
    printf("Invalid! Enter a positive number: ");
    scanf("%d", &num);
}

printf("You entered: %d\n", num);
Explanation:

The loop keeps asking for input until the user enters a positive number. If the first input is valid, the loop body never executes.

do-while Loop

The do-while loop is similar to the while loop, but the condition is checked after each iteration. This guarantees that the loop body executes at least once, even if the condition is initially false.

Syntax:
do {
    // Code to repeat
} while (condition);
Key Difference: while vs do-while
  • while: Condition checked before each iteration (may execute 0 times)
  • do-while: Condition checked AFTER each iteration (always executes at least 1 time)

Step-by-Step Execution:

1
Execute body: Run the code inside the loop (always runs first)
2
Check condition: Is the condition true?
3
If TRUE: Repeat from step 1
4
If FALSE: Exit the loop

Example: Print Numbers 1 to 5

int i = 1;

do {
    printf("%d\n", i);
    i++;
} while (i <= 5);

// Output:
// 1
// 2
// 3
// 4
// 5

Example: Menu System

int choice;

do {
    printf("\n=== Menu ===\n");
    printf("1. Option A\n");
    printf("2. Option B\n");
    printf("3. Exit\n");
    printf("Enter choice: ");
    scanf("%d", &choice);
    
    if (choice == 1)
        printf("You selected Option A\n");
    else if (choice == 2)
        printf("You selected Option B\n");
        
} while (choice != 3);

printf("Goodbye!\n");
Explanation:

The menu displays at least once. The loop continues until the user enters 3 to exit. This is a perfect use case for do-while because you want the menu to keep showing until customers decide to exit (enters 3)

break and continue Statements

These statements provide additional control over loop execution:

break

Immediately exits the loop, regardless of the loop condition. Execution continues with the first statement after the loop.

for (int i = 1; i <= 10; i++) {
    if (i == 5)
        break;  // Exit loop when i is 5
    printf("%d ", i);
}

// Output: 1 2 3 4

continue

Skips the rest of the current iteration and jumps to the next iteration. The loop does not exit.

for (int i = 1; i <= 5; i++) {
    if (i == 3)
        continue;  // Skip when i is 3
    printf("%d ", i);
}

// Output: 1 2 4 5

Example: Skip Even Numbers

printf("Odd numbers from 1 to 10:\n");

for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0)
        continue;  // Skip even numbers
    printf("%d ", i);
}

// Output: 1 3 5 7 9
Part 3

Advanced Repetition Structures

Types of Repetition Structures

Loops can be classified into three categories based on how they control repetition:

Counter-Controlled Repetition

The loop repeats a specific number of times. A counter variable tracks the number of iterations.

// Print "Hello" 5 times
for (int count = 1; count <= 5; count++) {
    printf("Hello\n");
}

// count is the counter variable
// Loop executes exactly 5 times
When to Use:

Use when you know in advance how many times to repeat (e.g., process 10 students, display 20 items, repeat N times).

Sentinel-Controlled Repetition

The loop continues until a special value (sentinel) is encountered. The sentinel signals the end of input.

// Calculate average until user enters -1
int num, sum = 0, count = 0;

printf("Enter numbers (-1 to stop):\n");
scanf("%d", &num);

while (num != -1) {  // -1 is the sentinel
    sum += num;
    count++;
    scanf("%d", &num);
}

if (count > 0)
    printf("Average: %.2f\n", (float)sum / count);

// Loop ends when user enters -1
When to Use:

Use when you don't know how many iterations are needed, but you have a special value that signals completion (e.g., -1, 0, "quit").

Event-Controlled Repetition

The loop continues until a specific event or condition occurs (not based on counting or a sentinel value).

// Keep trying until correct password
char password[20];
int attempts = 0;
int maxAttempts = 3;

do {
    printf("Enter password: ");
    scanf("%s", password);
    attempts++;
    
    if (strcmp(password, "secret") == 0) {
        printf("Access granted!\n");
        break;  // Event: correct password
    } else {
        printf("Wrong password\n");
    }
} while (attempts < maxAttempts);

if (attempts == maxAttempts)
    printf("Account locked\n");

// Loop ends when password is correct OR max attempts reached
When to Use:

Use when repetition depends on a condition or event (e.g., wait until data is ready, retry until success, read until end of file).

Nested Loops

A nested loop is a loop inside another loop. The inner loop completes all its iterations for each iteration of the outer loop.

Syntax:
for (initialization; condition; update) {
    // Outer loop body
    
    for (initialization; condition; update) {
        // Inner loop body
    }
    
    // Outer loop continues
}
How Nested Loops Work:
  • For each iteration of the outer loop, the inner loop runs completely
  • If outer loop runs 3 times and inner loop runs 4 times, total iterations = 3 × 4 = 12
  • You can nest any combination of for, while, and do-while loops

Example: Multiplication Table

// Print multiplication table (1 to 5)
for (int i = 1; i <= 5; i++) {        // Outer loop: rows
    for (int j = 1; j <= 5; j++) {    // Inner loop: columns
        printf("%d x %d = %d\t", i, j, i * j);
    }
    printf("\n");  // New line after each row
}

// Output:
// 1 x 1 = 1    1 x 2 = 2    1 x 3 = 3    1 x 4 = 4    1 x 5 = 5
// 2 x 1 = 2    2 x 2 = 4    2 x 3 = 6    2 x 4 = 8    2 x 5 = 10
// 3 x 1 = 3    3 x 2 = 6    3 x 3 = 9    3 x 4 = 12   3 x 5 = 15
// 4 x 1 = 4    4 x 2 = 8    4 x 3 = 12   4 x 4 = 16   4 x 5 = 20
// 5 x 1 = 5    5 x 2 = 10   5 x 3 = 15   5 x 4 = 20   5 x 5 = 25
Trace Through Execution:
  • Outer loop: i=1, Inner loop runs 5 times (j=1,2,3,4,5)
  • Outer loop: i=2, Inner loop runs 5 times again
  • ... continues until i=5
  • Total iterations: 5 × 5 = 25

Combining for and while

// Print star pattern
for (int i = 1; i <= 4; i++) {       // Outer: for loop
    int j = 1;
    while (j <= i) {                  // Inner: while loop
        printf("* ");
        j++;
    }
    printf("\n");
}

// Output:
// *
// * *
// * * *
// * * * *

Key Takeaways

  • Three control structures: sequence, selection, iteration
  • if statement executes code only if condition is true
  • if-else provides two paths: one for true, one for false
  • switch selects from multiple cases based on a value; use break to prevent fall-through
  • Null statement (;) does nothing
  • for loop: best when you know how many iterations (counter-controlled)
  • while loop: condition checked before each iteration (may execute 0 times)
  • do-while loop: condition checked after each iteration (always executes at least once)
  • break exits a loop immediately; continue skips to next iteration
  • Counter-controlled: fixed number of iterations
  • Sentinel-controlled: loop until special value encountered
  • Event-controlled: loop until specific condition/event occurs
  • Nested loops: loop inside another loop
  • Always update loop variables to avoid infinite loops
